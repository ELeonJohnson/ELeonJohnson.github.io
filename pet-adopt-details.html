Pet Adopt

PetAdopt is a full stack web application that I created. Users can register as an Owner to put their reptiles, cats or dogs up for adoption. Users can also register as an Adopter.

Tools: C#/ASP.NET Core, Sass, Bootstrap/CSS, HTML, Razor, SQL Server, User Authentication, User Roles, Linq Queries, User Identity, Media Queries
